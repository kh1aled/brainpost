# BrainPost **BrainPost — A Modern Full-Stack Blogging Platform** **BrainPost** is a full-stack blogging platform built with **Next.js 15**, **MySQL**, and **Sass**, designed to deliver a fast, secure, and seamless experience for both readers and creators. It allows users to share ideas, express creativity, and interact with posts through comments and likes — all in one clean and responsive interface. With built-in authentication, profile management, and real-time interactivity, BrainPost provides everything a modern content-sharing platform needs. **Project Owner / Team:** `[khaled hamdy]` **Start Date:** `[5 / 11]` **Main Goal / Vision:** To build a **modern full-stack blogging platform** where users can **create, interact, and manage content seamlessly**, combining **Next.js performance** with a**clean, responsive UI**. The goal is to deliver a **complete end-to-end solution** that includes **authentication, posts, likes, comments, and profile management**, all connected to a secure **MySQL database** and deployed for public use.

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).
